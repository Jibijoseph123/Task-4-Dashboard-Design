# Task 4: Dashboard Design

## Objective
Designed an interactive sales dashboard for business stakeholders.

## Tool Used
- Power BI

## Dataset
- Superstore Sales Dataset

## Features
- KPI cards for Sales, Profit, and Orders
- Time-series sales trend analysis
- Category and regional performance insights
- Interactive slicers and filters
- Clean and consistent visual theme

## Outcome
Created a stakeholder-ready dashboard that supports business decision-making.
